# easy-load-test-plan

## Build:
```bash
sh build.sh
```

## Run Binary (Unix)
```bash
./eztest.linux32 plan.yml
./eztest.linux64 plan.yml
./eztest.mac64 plan.yml
```

## Run Binary (Win)
```bash
eztest.win32.exe plan.yml
eztest.win64.exe plan.yml
```

## Just Go Run:
```bash
go run ez-test.go plan.yml
```
